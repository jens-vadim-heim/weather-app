# Weather app

## What?
This is a simple weather web app where the user can look up the weather at a specific location. 

Look up weather information by typing a location in the search bar, then press the "Go" button or just the enter-button to fetch and display the data.

## How?

The app is made using HTML, vanilla CSS and Javascript/React (built using Vite). It fetches weather data from the OpenWeather API.
